#3 index.htmlを作成し、gits-takaoka.github.io で公開できる

why

  • github上に格納したindex.htmlを、gitst-takaoka.github.io/first-sample/index.html で公開したい
以下の開発フローを実施できるようになりたい
  • 作業開始の起点となるIssue を発行する
  • 発行されたIssue番号をもとに、ブランチを作成する
  • ブランチ内で ファイルを編集し、git commit する
  • git commit した ブランチを git push する
  • git push したブランチを、masterブランチにマージするPull Request を作成する
  • 作成したPullRequestのレビューを依頼する
  • 作成したPullRequestがマージされる、つまり、master ブランチにマージされる

What

  • #3 の issue を作成した
  • GitKraken で、#3-index.html ブランチを作成する
  • #3-index.html ブランチ で、index.html ファイルを編集し、git commit する
  • #3-index.html ブランチ を GitKraken で、push する
    　github 上で、#3-index.html ブランチ　がpushされたことを確認する
  • #3-index.html ブランチ を master ブランチにマージするための PullRequest を作成する
